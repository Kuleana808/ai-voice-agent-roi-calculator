# AI Voice Agent ROI Calculator

Static GitHub Pages site that helps local service businesses estimate how much revenue an AI voice agent could recover from missed calls.

- Plumbers
- HVAC companies
- Electricians
- Roofers
- Dentists
- Med spas
- Urgent care clinics
- Physical therapy clinics
- Ophthalmology clinics
- Dermatology clinics
- Personal injury law firms
- Orthopedic clinics
- Plastic surgery practices
- Fertility clinics
- Oral surgery practices
- Cosmetic dentistry practices
- Other call-driven local businesses

## Conversion goal
Warm inbound leads via prefilled `mailto:` CTA to `leads@operator.fyi`.
